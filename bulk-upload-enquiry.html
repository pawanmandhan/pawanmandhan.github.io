Bulk Upload Enquiry

User Story

Bulk upload of enquiry will be a new page. Navigation link in NavBar for this page will have name - "Upload All Enquiry". User should be able to download format of excel from this page. User should be able to upload excel sheet containing multiple enquiries.

Functioning of UI

  • "Download Excel Format" button will be used to download excel format template of enquiries.
  • On hovering over "Download Excel Format" button, a tooltip should be displayed with message "Download excel format of enquires".
  • Upload button should upload a excel file from client's machine to webpage.
  • How to select location?
    1. To select district - Select state and then select corresponding district to create a pair
    2. To select whole state - Select only State
    3. Step 1/2 can be repeated multiple times to create more combinations of state and corresponding districts.
  • How to unselect a State district pair?
    • Click on cross icon on the pair of state district.
  • Submit button should be enabled only when all validations are passed.
  • Reset button will reset all fields in form.

Validations

  • Upload should only upload excel files. Check different extensions of excel file for this purpose.
  • Upload should be allowed for only one file at a time.
  • State
    • Optional
    • Required to select district
  • District
    • Optional (no matter state is selected or not).
  • Remark - alphanumeric(text) - (Compulsory) - min 10, max 100 char

On Click of Save Button

  • If excel file is valid and all fields in it are also valid.
    • Create Project if not present
    • Create Enquiry
    • if enquiry is created successfully
    • send email and notification(on ui) to all sellers who has the products listed in enquiry.
    • To find the sellers who has the product
    • Find by text search on product name and product description.
    • Filter seller by location specified in enquiry and the location of seller.
    • if enquiry creation fails
      • show error message in toastr ("Failed to create Enquiry. Please try again")
      • User should be allowed to again save the enquiry with all form fields intact.
  • If Enquiry No was empty (means new enquiry)
    • Create Project if not present
    • Create Enquiry
    • if enquiry is created successfully
      • send email and notification(on ui) to all sellers who has the products listed in enquiry.
      • To find the sellers who has the product
        • Find by text search on product name and product description.
        • Filter seller by location specified in enquiry and the location of seller.
    • if enquiry creation fails
      • If excel format is not correct
        • show error message in toastr ("Failed to create Enquiry. Excel is not correct")
      • if failed due to backend issue
        • show error message in toastr ("Failed to create Enquiry due to temporary issue. Please try again later.")
      • User should be allowed to again save the enquiry with all form fields intact.
